# Personal Notes Application

A personal notes application built with React + Vite.

[TBA] - not deployed yet.

## ✨ Features
- **CRUD Notes**: Create, delete, and archive notes
- **Search**: Real-time search notes by title
- **Archive/Unarchive**: Separate active and archived notes sections
- **Character Limit**: 50 character limit for note titles with remaining count display
- **Responsive Design**: Clean UI on mobile and desktop
- **Empty States**: Informative messages when notes/archive are empty
- **Date Formatting**: Creation date displayed in localized Indonesian format

## 🛠️ Tech Stack
- **React 18** - UI Library
- **Vite** - Lightning fast build tool
- **Vanilla CSS** - No CSS frameworks, pure styling
- **date-fns** - Date formatting utility

## ⚙️ How to Run
```bash
# 1. Install dependencies

# 2. Start development server

# 3. Build for production
